# Pitagoras

Pitagoras es un calculador de longitudes con el teorema de pitagoras

## Git clone 🚀

_$git clone https://github.com/mkyvent/pitagotas.git._


### Pre-requisitos 📋

_Golang 1.19.2_

```
  Linux:
     apt install golang -y
  Termux:
     pkg install golang -y
```

### Instalación 🔧

_go run pitagoras.go_
